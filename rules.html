Rules:-

  • Gifts are hidden at random by the computer on the grid.
  • The game has three levels: Easy(3x3 grid) , Medium(4x4 grid) and Difficult(6x6 grid)
  • The game will prompt for a guess
  • You have to entr the row and column no in the input box . for example if you think the gift is in 1st row and 1st column type "A1" ans similary for others.
  • Submit your guess
  • If it is a gift find all other as well.
  • If it is a miss then you need to go for next guess
  • And the game is over as you come on a danger symbol or make run out of moves.
